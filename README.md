# tikz-control-systems
Control theory blocks using PGF/TikZ

The controlsys.tex contains the implementation of the blocks. 
Currently, a saturation and a rate limiter block are implemented. 

The example_controlsys.tex presents a use case example.
Run the example using "pdflatex example_controlsys.tex"
